一个Git仓库下有三个不同的区域：
Workspace(工作区)， Stage或者Index(暂存区)、Respository（仓库区）
我们针对项目代码的更改操作都是在工作区进行的，通过执行git add *命令就把在工作区的改动提交到了暂存区，通过执行git commit -m ‘info about this commit’就把已经存放在暂存区的改动给提交到了当前分支的仓库区
